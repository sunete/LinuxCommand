write

向指定登录用户终端上发送信息

补充说明

write命令 用于向指定登录用户终端上发送信息。通过write命令可传递信息给另一位登入系统的用户，当输入完毕后，键入EOF表示信息结束，write命令就会将信息传给对方。如果接收信息的用户不只登入本地主机一次，你可以指定接收信息的终端机编号。

语法

write(参数)

参数

用户：指定要接受信息的登录用户；
登陆终端：指定接收信息的用户的登录终端。

实例

传信息给Rollaend，此时Rollaend只有一个连线 : 

write Rollaend
